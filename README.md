# ハクレイフロンティア カード検索

ハクレイフロンティアのカードデータベースをブラウザで検索できるWebアプリケーションです。

## 🌐 デモ

https://ikeda-8871.github.io/kamiha/

## ✨ 機能

- **カード名・タグ検索**: カード名やタグで検索
- **効果テキスト検索**: カードの効果テキスト内を検索
- **詳細フィルタ**:
  - タイプ別フィルタ (キャラクター/能力)
  - パック別フィルタ (複数選択可能)
  - タグによる絞り込み
- **表示件数制限**: 結果の表示件数を調整可能
- **Q&A表示**: カードの質問と回答を表示
- **レスポンシブデザイン**: モバイル・タブレット・デスクトップに対応

## 🔍 検索の使い方

- **カード名・タグ検索**: 「霊夢」「妖精」などを入力
- **効果テキスト検索**: 「無効」「ドロー」などの効果キーワードを入力
- **タグフィルタ**: 複数のタグで絞り込む場合はカンマ区切り (例: `紅魔館,妖怪`)
- **パック選択**: Ctrlキー (Mac: Command) を押しながらクリックで複数選択
